# SmartSupport Landing Page

Стильная landing страница для проекта SmartSupport - интеллектуального помощника по работе с обращениями граждан.

## 🎯 Описание

Landing страница представляет проект SmartSupport, разработанный командой "Имени товарища Вертера" (КИТВ) от Школы 21 для участия в кейс-чемпионате Белгородской области в сфере информационных технологий (ИТ-Спринт 2025).

## ✨ Особенности

- **Современный дизайн** с использованием Tailwind CSS
- **Адаптивная верстка** для всех устройств
- **Интерактивные элементы** с плавными анимациями
- **SEO-оптимизация** с мета-тегами
- **Быстрая загрузка** с оптимизированными ресурсами

## 📋 Содержание

### Основные разделы:

1. **Hero Section** - Главный экран с описанием проекта
2. **О проекте** - Подробное описание SmartSupport
3. **Возможности** - Функциональность системы с ссылками на интерфейсы
4. **Технологии** - Технологический стек с иконками
5. **Видение будущего** - Планы развития проекта
6. **Демо** - Ссылки на интерактивные интерфейсы

### Ключевые особенности:

- **Информация о команде** - КИТВ от Школы 21
- **Описание MVP** - Текущая версия с ограниченным функционалом
- **Технические детали** - Реализация базы знаний и поиска
- **Коммерческое применение** - Потенциальные клиенты и преимущества

## 🛠 Технологии

- **HTML5** - Семантическая разметка
- **Tailwind CSS** - Современная CSS-фреймворк
- **JavaScript** - Интерактивность и анимации
- **Font Awesome** - Иконки
- **Google Fonts** - Типографика (Inter)
- **Nginx** - Веб-сервер

## 🚀 Деплой

### Автоматический деплой:

```bash
# Из корня проекта
./deploy-landing.sh
```

### Ручной деплой:

```bash
# Сборка и запуск
docker-compose -f docker-compose.production.yml build landing
docker-compose -f docker-compose.production.yml up -d landing

# Проверка статуса
docker ps | grep smart-support-landing
docker logs smart-support-landing
```

## 🎨 Дизайн-система

### Цветовая палитра:
- **Primary:** Градиент от синего к фиолетовому
- **Background:** Светло-серый (#f9fafb)
- **Text:** Темно-серый (#374151)
- **Accent:** Синий (#3b82f6)

### Типографика:
- **Font Family:** Inter (Google Fonts)
- **Headings:** 700-800 weight
- **Body:** 400-500 weight

### Анимации:
- **Float:** Плавное покачивание для hero-элементов
- **Hover:** Поднятие карточек при наведении
- **Scroll:** Появление элементов при прокрутке

## 🔧 Настройка

### Изменение контента:

1. Отредактируйте `index.html`
2. Пересоберите контейнер:
   ```bash
   docker-compose -f docker-compose.production.yml build landing
   docker-compose -f docker-compose.production.yml up -d landing
   ```

### Настройка Nginx:

Конфигурация находится в `nginx-configs/smartsupport.vadimevgrafov.ru.conf`

### SSL сертификаты:

Автоматически генерируются через Let's Encrypt при первом запуске.
